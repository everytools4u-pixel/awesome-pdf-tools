# 📄 Awesome PDF Tools

The best free online PDF tools. No installation needed.

## Convert
- [PDF to Word](https://the-pdf.com/tools/pdf-to-word/) - Convert PDF to DOCX
- [Word to PDF](https://the-pdf.com/tools/word-to-pdf/) - Convert DOCX to PDF
- [PDF to JPG](https://the-pdf.com/tools/pdf-to-jpg/) - Extract images from PDF
- [Excel to PDF](https://the-pdf.com/tools/excel-to-pdf/) - Spreadsheet to PDF

## Edit
- [Merge PDF](https://the-pdf.com/tools/merge-pdf/) - Combine multiple PDFs
- [Split PDF](https://the-pdf.com/tools/split-pdf/) - Split PDF into pages
- [Compress PDF](https://the-pdf.com/tools/compress-pdf/) - Reduce file size
- [Rotate PDF](https://the-pdf.com/tools/rotate-pdf/) - Rotate PDF pages

## Security
- [Protect PDF](https://the-pdf.com/tools/protect-pdf/) - Add password protection
- [Unlock PDF](https://the-pdf.com/tools/unlock-pdf/) - Remove PDF password
